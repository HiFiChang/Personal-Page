## 为什么要写博客？

写作即思考。我一直觉得，将想法转化为文字的过程，能迫使我达到一种纯粹思考所无法达到的清晰度。这个博客是我分享学习、创造和思考的空间。

## 关于我

我是**中国人民大学**计算机科学与技术专业的大三学生。每天的日常大概是在读论文、写代码、偶尔盯着终端输出思考到底哪里出了问题之间循环。

我的兴趣涵盖以下几个方面：

- **人工智能** — 尤其是大语言模型及其应用
- **系统编程** — 编译器、操作系统，以及底层代码的美感
- **创意技术** — 做既实用又好看的东西

## 未来的内容

我计划写一些这样的文章：

1. 正在做的项目的技术深入分析
2. 论文阅读与思考

## 一段代码

一篇第一篇博客怎能没有代码呢：

```python
def greet(name: str) -> str:
    """一句温暖的问候。"""
    return f"你好，{name}。欢迎来到博客。"

if __name__ == "__main__":
    print(greet("世界"))
```
